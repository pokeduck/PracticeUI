# PracticeUI
The exercise about UIKit

## Install CLI Tools

1. ``brew install mint``
2. ``mint bootstrap --link``
3. ``brew install rbenv``
4. ``rbenv local``
5. ``gem install bundle``
6. ``bundle install``

## Generate .xcodeproj & .xcworkspace
1. ``carthage bootstrap``
2. ``xcodegen``
3.  ``pod install``
4. Open ``PracticeUI.xcworkspace``
